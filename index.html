How to compress, extract files with tar in terminal

To Zip, Unzip

zip -r name.zip folder_to_compress

unzip name.zip

tar.gz

tar -zcvf name.tar.gz folder-to-compress

tar -zxvf name.tar.gz

x is extract

c is compress

v is verbose, which show the file names

f mean the next argument is filename

z mean tar.gz file type

j mean tar.bz2
